## Live

- **Share this:** https://kerf-landing.vercel.app/
- Vercel project: Katalyst team `kerf-landing`
- Repo: https://github.com/KatTate/kerf-landing
- GitHub Pages (office filter may block): https://kattate.github.io/kerf-landing/

## Local

Open `index.html` or serve the folder:

```bash
python3 -m http.server 8765
```

## Deploy

```bash
npx vercel@latest deploy --prod --yes --scope katalyst-c6ba0763
```

Static site. GitHub repo is linked to the Vercel project.
